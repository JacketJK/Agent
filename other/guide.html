Samitivej Hospital

เริ่มต้นสะสม Point ง่ายๆ

เรียนรู้วิธีสะสม Point เพื่อรับสิทธิประโยชน์มากมายจาก Samitivej Hospital ผ่านการใช้งานแอปพลิเคชันและการเข้าร่วมกิจกรรมต่างๆ

วิธีสะสม Point
1
ลงทะเบียนใช้งานแอป

สร้างบัญชีผู้ใช้ในแอปพลิเคชัน Samitivej

2
เข้ารับบริการทางการแพทย์

ทุกครั้งที่เข้ารับบริการจะได้รับ Point ตามมูลค่าการใช้บริการ

3
เข้าร่วมกิจกรรมและโปรโมชั่น

ติดตามกิจกรรมพิเศษที่ให้ Point เพิ่มเติม

4
เชิญเพื่อนและครอบครัว

รับ Point เมื่อเพื่อนที่เชิญมาใช้งานแอป

เคล็ดลับการสะสม Point
  • ตรวจสอบ Point ของคุณเป็นประจำในแอปพลิเคชัน
  • เข้าร่วมกิจกรรมและโปรโมชั่นพิเศษ
  • ใช้บริการในเครือข่าย Samitivej เพื่อรับ Point เพิ่มเติม
  • ติดตามข่าวสารและอัปเดตจากแอปพลิเคชัน
คำถามที่พบบ่อย
Point มีอายุการใช้งานหรือไม่?
Point มีอายุการใช้งาน 2 ปี นับจากวันที่ได้รับ
สามารถโอน Point ให้ผู้อื่นได้หรือไม่?
Point ไม่สามารถโอนให้ผู้อื่นได้ แต่สามารถใช้เพื่อซื้อของขวัญหรือบริการให้ผู้อื่นได้
ต้องสะสม Point ถึงจำนวนเท่าไรจึงจะใช้ได้?
สามารถใช้ Point ได้ตั้งแต่ 100 Point ขึ้นไป
วิธีตรวจสอบ Point ที่สะสมได้?
เข้าไปที่เมนู "My Points" ในแอปพลิเคชันเพื่อดู Point ที่สะสมได้
ต้องการความช่วยเหลือ?

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ สามารถติดต่อทีมสนับสนุนได้
