เริ่มต้นใช้งานผ่าน LINE

เรียนรู้วิธีใช้งานแอปพลิเคชัน Samitivej ผ่าน LINE LIFF เพื่อการดูแลสุขภาพที่สะดวกและเชื่อมต่อกับเพื่อนใน LINE ได้ง่าย

LINE LIFF คืออะไร?

LINE LIFF (LINE Front-end Framework) เป็นแพลตฟอร์มที่ช่วยให้คุณใช้งานแอปพลิเคชันต่างๆ ได้โดยตรงผ่าน LINE โดยไม่ต้องดาวน์โหลดแอปแยก ทำให้สะดวกและประหยัดพื้นที่อุปกรณ์

คู่มือการใช้งานแบบขั้นตอน
1
การเชื่อมต่อกับ LINE

เข้าสู่ระบบผ่าน LINE ID ของคุณ

  • แอปจะขอสิทธิ์เข้าถึงข้อมูล LINE Profile
  • ยืนยันการอนุญาตเพื่อเชื่อมต่อบัญชี
  • ระบบจะดึงข้อมูล LINE Profile อัตโนมัติ
2
การลงทะเบียนและยืนยันตัวตน

กรอกข้อมูลเพิ่มเติมและยืนยันตัวตน

  • กรอกข้อมูลส่วนตัวที่จำเป็น
  • อัปโหลดรูปถ่ายบัตรประชาชน
  • ยืนยันหมายเลขโทรศัพท์
  • รอการอนุมัติจากระบบ (24-48 ชั่วโมง)
3
การใช้งานฟีเจอร์หลัก

เข้าถึงบริการต่างๆ ผ่าน LINE

  • จองคิวแพทย์และนัดหมาย
  • ดูประวัติการรักษา
  • ชำระเงินออนไลน์
  • สะสม Point และรับสิทธิประโยชน์
4
การตั้งค่าและการแจ้งเตือน

ปรับแต่งการใช้งานตามความต้องการ

  • เปิด/ปิดการแจ้งเตือน LINE
  • ตั้งค่าการแจ้งเตือนการนัดหมาย
  • เลือกภาษาและรูปแบบการแสดงผล
5
การใช้งานร่วมกับเพื่อน

แชร์และเชิญเพื่อนใช้งานแอป

  • แชร์ข้อมูลการนัดหมายกับครอบครัว
  • เชิญเพื่อนใช้งานแอปเพื่อรับ Point
  • ส่งข้อความผ่าน LINE Chat

ความปลอดภัยสูง

ใช้ระบบความปลอดภัยของ LINE

เคล็ดลับการใช้งาน LINE LIFF
  • เปิดการแจ้งเตือน LINE เพื่อไม่พลาดข้อมูลสำคัญ
  • ใช้ฟีเจอร์แชร์เพื่อส่งข้อมูลให้ครอบครัว
  • เชิญเพื่อนใช้งานเพื่อรับ Point เพิ่มเติม
  • ตรวจสอบข้อมูลการนัดหมายเป็นประจำ
  • ใช้ LINE Chat เพื่อติดต่อกับทีมสนับสนุน
ข้อควรระวัง
  • ต้องมีบัญชี LINE และเชื่อมต่ออินเทอร์เน็ต
  • ข้อมูลส่วนตัวจะถูกใช้ตามนโยบายความเป็นส่วนตัว
  • ไม่ควรแชร์ข้อมูลการรักษากับผู้อื่นโดยไม่ได้รับอนุญาต
  • ติดต่อทีมสนับสนุนหากพบปัญหาในการใช้งาน
การแก้ไขปัญหาที่พบบ่อย

  • ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
  • ตรวจสอบว่าเปิดแอป LINE แล้ว
  • ลองปิดและเปิดแอป LINE ใหม่
  • ตรวจสอบเวอร์ชัน LINE ให้เป็นเวอร์ชันล่าสุด

  • ตรวจสอบการตั้งค่าการแจ้งเตือนใน LINE
  • ตรวจสอบการตั้งค่าในอุปกรณ์
  • ลองปิดและเปิดการแจ้งเตือนใหม่
  • ตรวจสอบว่าไม่ได้บล็อก Official Account

  • ตรวจสอบความเร็วอินเทอร์เน็ต
  • ปิดแอปอื่นที่ไม่ใช้งาน
  • ลองรีเฟรชหน้าเว็บ
  • ตรวจสอบพื้นที่ว่างในอุปกรณ์
ต้องการความช่วยเหลือเพิ่มเติม?
